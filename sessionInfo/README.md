# Session information

This folder contains R session records for dataset-level upstream processing
and the stable core statistical modules.

The core module session records correspond to the successful final runs used to
produce the Route C submission source tables:

- `01_ubl3_donor_level_endpoints_sessionInfo.txt`
- `02_psp_v1_robustness_sensitivity_sessionInfo.txt`
- `03_pseudobulk_deseq2_output_sessionInfo.txt`

Figure layout sessionInfo files are intentionally omitted because final figure
composition scripts are not part of the stable core analysis archive.
