# UBL3 expression across tauopathies: reproducibility code

This repository contains the analysis code for the manuscript on **UBL3
expression in single-cell / single-nucleus RNA-seq data across tauopathies**.

The study analyzes 4 included public datasets, corresponding to 7 donor-level
disease-region analytical units:

- GSE157827: Alzheimer disease, middle frontal gyrus
- GSE174367: Alzheimer disease, prefrontal cortex
- syn21788402: Alzheimer disease, entorhinal cortex and superior frontal gyrus
- syn52082747: Alzheimer disease, progressive supranuclear palsy, and
  Pick's disease / frontotemporal-lobar degeneration in primary visual cortex

Two screened CTE datasets were considered during dataset screening but were not
included in the inferential analysis set.

## Scope of this code archive

This GitHub repository is intended to reproduce:

1. the harmonized processed Seurat objects used by the manuscript analyses;
2. the core donor-level statistical analyses;
3. the supplementary tables and source-data files underlying the manuscript.

Final journal-formatted figures, very large processed objects, and large
spreadsheet outputs are not stored in this GitHub repository. They are provided
through the manuscript additional files and/or the associated Zenodo data
deposit.

## Code organization

### 01_per_dataset

These scripts reproduce the harmonized `celltype6` processed objects used by
the integrated analyses.

- `GSE157827/`: raw 10x data to harmonized AD/control object
- `GSE174367/`: raw h5 data to harmonized AD/control object
- `syn21788402/`: author-provided SCE objects to EC and SFG harmonized objects
- `syn52082747/`: author-provided object to the harmonized AD/PSP/FTD/control
  object `stepH_slim_uncompressed.rds`

### 02_core_statistics

These scripts generate the numerical results used for the manuscript's main
claims and supplementary tables.

- `S1_detection_breadth.R`: UBL3 donor-level detection-breadth and conditional
  expression analyses across the 7 analytical units
- `S2_housekeeping_baseline.R`: housekeeping-gene baseline analysis
- `S3_SUMO_comparator.R`: SUMO1/SUMO2/SUMO3 comparator analysis
- `S4_robustness/`: FDR sensitivity, permutation, leave-one-donor, and manual
  recomputation checks for the focal findings
- `Data1_pseudobulk_DESeq2.R`: full-transcriptome donor-level pseudobulk DESeq2
  output used for Supplementary Data 1

The final figure layout scripts used to create journal-formatted 170-mm BMC
figures are not part of the minimal code archive because they are formatting
scripts rather than primary analysis scripts. The numerical source data and
statistical outputs behind those figures are generated by the scripts above.

## Data availability

Raw single-cell / single-nucleus RNA-seq data are publicly available from the
accessions listed in `metadata/data_sources.tsv`.

Large derived Seurat objects and large spreadsheet outputs are not stored in
GitHub. They should be deposited in Zenodo or another persistent public data
repository and cited in the manuscript Availability of data and materials
section.

## Reproducibility notes

- Random seeds are script-specific and are declared near the top of each script. Most integrated statistical scripts use 20251023; syn21788402 object reconstruction uses SEED = 42 to match the original upstream code.
- R session information from successful runs is stored in `sessionInfo/`.
- Checksums for large processed objects should be recorded in
  `metadata/processed_objects_manifest.tsv` and in the Zenodo deposit.
